# kaimon 個別相談 分析ダッシュボード

GitHub Pages で公開する分析ツールです。  
Google スプレッドシートのデータを可視化します。

---

## 機能

| 機能 | 内容 |
|------|------|
| サマリーカード | 総相談数・入会数・転換率・検討中件数 |
| 月別推移グラフ | 相談数/入会数(棒) + 転換率(折れ線) |
| 担当者別実績 | 担当者ごとの相談数・入会数・転換率 |
| 撮影ジャンル別 | 入会者のジャンル分布（ドーナツグラフ）|
| 年齢別グラフ | 年齢帯ごとの相談・入会数 |
| 今月の状況 | 今月集計 + 期限まで30日以内の検討中一覧 |
| 個別データ一覧 | 入会状況フィルター・名前検索・期限アラート |

## セットアップ手順

### STEP 1 : GAS を設定する

1. スプレッドシートのメニュー **「拡張機能」→「Apps Script」** を開く
2. 既存コードを全削除し、**`gas_code.gs`** の内容を丸ごと貼り付け
3. **「デプロイ」→「新しいデプロイ」** をクリック
4. 種類 → **「ウェブアプリ」**
5. 実行ユーザー → **「自分（メールアドレス）」**
6. アクセス権限 → **「全員」**
7. 「デプロイ」→ 権限を許可
8. 表示された **ウェブアプリURL** をコピー（`https://script.google.com/macros/s/xxx/exec` の形式）

### STEP 2 : GitHub Pages に公開

1. このフォルダを GitHub リポジトリとして作成
2. Settings → Pages → Branch: `main` / `root` で公開
3. 公開 URL（`https://yourname.github.io/kaimon-analytics/`）にアクセス

### STEP 3 : GAS URL を設定

1. ダッシュボードを開く
2. 右上の **「設定」ボタン** をクリック
3. STEP 1 でコピーした URL を貼り付けて **「保存して更新」**
4. URLは `localStorage` に保存されるので次回以降は不要

---

## GAS URL を更新した場合

スプレッドシートの構成変更等でGASを再デプロイすると URL が変わります。  
その場合は設定画面から新しい URL に更新してください。

---

## データ入力について

データ入力は引き続き **Google スプレッドシート** で行います。  
ダッシュボードはリードオンリーです（閲覧・分析のみ）。

「入会」列には以下の値を入力してください：
- `入会` — 入会確定
- `検討中` — 検討中（「検討中の場合の期限」に日付を入れると期限アラートが表示されます）
- `見送り` — 見送り
